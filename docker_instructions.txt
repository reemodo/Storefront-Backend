1. Ensure .env exists at repository root with the variables (see
backend/.env.example)
2. Start containers: docker compose up -d
3. Create test DB: docker exec -it <postgres_container> psql -U postgres -c
"CREATE DATABASE store_test;"
4. Run migrations: docker compose exec backend yarn migrate
5. Start backend: docker compose exec backend yarn dev (or run locally: cd
backend && yarn dev)
